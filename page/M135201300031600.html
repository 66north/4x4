Code No.11: Abnormality in FR Wheel Speed Sensor Circuit (Open Circuit)<br />Code No.12: Abnormality in FL Wheel Speed Sensor Circuit (Open Circuit)<br />Code No.13: Abnormality in RR Wheel Speed Sensor Circuit (Open Circuit)<br />Code No.14: Abnormality in RL Wheel Speed Sensor Circuit (Open Circuit)














OPERATION

  • A toothed rotor generates a voltage signal as it moves across the pickup field of each wheel speed sensor.
  • The amount of voltage generated at each wheel is determined by the clearance between the rotor teeth and the wheel speed sensor, and by the speed of rotation.
  • The wheel speed sensors transmit the frequency of the voltage signal to the ABS-ECU.

DIAGNOSIS CODE SET CONDITIONS
Diagnosis codes No.11, 12, 13 and 14 are set when sensor impedance is varies due to breakage of the wiring harness, connectors or coil of the four wheel speed sensors.

PROBABLE CAUSES

  • Malfunction of the wheel speed sensor
  • Damaged wiring harness or connector
  • Malfunction of the ABS-ECU

DIAGNOSIS

When the code No.11 is set

When the code No.12 is set
  • Item No.12: Front-left wheel speed sensor
When the code No.13 is set
  • Item No.13: Rear-right wheel speed sensor
When the code No.14 is set
  • Item No.14: Rear-left wheel speed sensor

Q. Is the check result normal?
Go to Step 12.
Go to Step 2.

STEP 2. Resistance measurement at D-256 ABS-ECU connector.
(1)Disconnect the D-256 ABS-ECU connector, and then measure the resistance at the harness connector side.
(2)Measure the following resistance between the terminals.
When the code No.11 is set
  • Between the D-256 ABS-ECU connector terminal No.10 and No.23
When the code No.12 is set
  • Between the D-256 ABS-ECU connector terminal No.7 and No.20
When the code No.13 is set
  • Between the D-256 ABS-ECU connector terminal No.8 and No.21
When the code No.14 is set
  • Between the D-256 ABS-ECU connector terminal No.9 and No.22
Standard Value: 1.0 - 1.5 kΩ

Q. Is the check result normal?
Go to Step 12.
NO (When resistance between the D-256 ABS-ECU connector terminal No.10 and No.23 is not within the standard value.) : Go to Step 3.
NO (When resistance between the D-256 ABS-ECU connector terminal No.7 and No.20 is not within the standard value.) : Go to Step 4.
NO (When resistance between the D-256 ABS-ECU connector terminal No.8 and No.21 is not within the standard value.) : Go to Step 5.
NO (When resistance between the D-256 ABS-ECU connector terminal No.9 and No.22 is not within the standard value.) : Go to Step 6.

STEP 8. Check the continuity between D-256 ABS-ECU connector and A-36 wheel speed sensor (FR) connector.

(2)Check the continuity between the following terminals.

  • D-256 ABS-ECU connector terminal No.10 - A-36 wheel speed sensor (FR) terminal No.1

OK: Continuity exists (2 Ω or less)

STEP 12. Check whether the diagnosis code is reset.
Check again if the diagnosis code is set.
(1)Drive the vehicle with speed exceeds 8 km/h to recognize wheel signals to the ABS-ECU.
(2)Erase the diagnosis code.
(3)Turn the ignition switch to the "LOCK" (OFF) position.
(4)Turn the ignition switch to the "ON" position.
(5)Disconnect the M.U.T.-III.
(6)Drive the vehicle with speed exceeds 40 km/h to recognize wheel signals to the ABS-ECU.
(7)Check if the diagnosis code is reset.
(8)Turn the ignition switch to the "LOCK" (OFF) position.

Q. Is code No.11, 12, 13 or 14 set?
