GENERAL INFORMATION


The engine immobilizer system prevents the engine from starting and immobilizes the vehicle if a key other than the key registered for that vehicle is used in an attempt to start the engine after forced entry. The engine immobilizer system consists of the ignition key, key ring antenna, immobilizer-ECU <Diesel powered vehicle> or immobilizer-ECU with an antenna <Petrol powered vehicle> and the engine-ECU <M/T> or engine-A/T-ECU <A/T>. It works in the following way and has these functions.

  1. With the ignition key turned ON, the transponder (a small transmitter) integrated in the ignition key transmits its own key ID to the key ring antenna via radio wave <Diesel powered vehicle> or the ring antenna via radio wave <Petrol powered vehicle>.
  2. According to the sent key ID, the immobilizer-ECU controls the engine-ECU <M/T> or engine-A/T-ECU <A/T> only when the sent key ID agrees with the pre-registered one.
  3. The system is designed to be maintenance-free because the power source for the transponder is supplied by the immobilizer-ECU. Two ignition keys are provided, and up to eight keys can be registered to one vehicle as needed. More than one trillion of key ID combinations can be registered. This feature prevents code copying, resulting in higher security of the system.

<Diesel powered vehicle>
Component name
Outline of function
Transponder
 Is power-supplied by the key ring antenna. When the transponder receives random number data, it processes it and the key ID. Then it transmits the process result to the immobilizer-ECU.
Key ring antenna
 The immobilizer-ECU supplies power via an antenna on a steering lock by transmission of electromagnetic waves to a transponder built into a key, using magnetic coupling
Immobilizer-ECU
  • Supplies electrical power to the transponder integrated in the ignition key, and transmits random number data.
  • Verifies the key ID which is sent from the transponder. If the code is correct, it sends an engine mobilization signal to the engine-ECU.

Engine-ECU <M/T> or engine-A/T-ECU <A/T>
 Starts the engine, and then continues the engine running if an engine mobilization signal is confirmed. If an engine immobilization signal is confirmed, the ECU cancels the engine control and stops the engine.

caution
  • Do not replace the engine-ECU <M/T> or engine-A/T-ECU <A/T> and the immobilizer-ECU simultaneously. Always replace the ECU by ones when the multiple ECU is replaced, and then replace the next ECU after registering the necessary IDs.
  • Do not register the ignition key registered for other vehicle.
