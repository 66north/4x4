Code No.11, 12: Power supply voltage system




















OPERATION


When turning the ignition switch to ON, the transfer-ECU (terminal No.13) receives system voltage from the ignition switch.

DIAGNOSIS CODE SET CONDITIONS


When power supply voltage drops to 9.5 V and less, the code No.11 is set as low voltage.
When power supply voltage rises to 18 V and over, the code No.12 is set as high voltage.

PROBABLE CAUSES



  • Malfunction of the ignition switch
  • Damaged harness wires and connectors
  • Malfunction of the transfer-ECU

DIAGNOSIS PROCEDURE

STEP 1. M.U.T.-III data list
Item 9: Ignition switch power supply (Refer to data list reference table ).

Q. Is the check result normal?
Intermittent malfunction (Refer to GROUP 00 - How to Cope with Intermittent Malfunction ).
Go to Step 2.

STEP 2. Measure the voltage at transfer-ECU connector D-124.
(1)Turn the ignition switch to the ON position.
(2)Measure the voltage between transfer-ECU connector D-124 terminal No.13 and earth.
OK: System voltage

Q. Is the check result normal?
Go to Step 5.
Go to Step 3.

STEP 3. Connector check: D-34 J/C (4), D-403, D-405 J/B connector, D-228 intermediate connector, D-124 transfer-ECU connector.
Check for the contact with terminals.

Q. Is the check result normal
Go to Step 4.
Repair the defective connector.

STEP 4. Check the harness between J/B connector D-403 terminal No.2 and transfer-ECU connector D-124 terminal No.13.
Check the power supply line for short or open circuit.

Q. Is the check result normal?
Go to Step 6.
Repair the wiring harness.

STEP 5. Connector check: D-124 transfer-ECU connector.
Check for the contact with terminals.

Q. Is the check result normal?
Go to Step 6.
Repair the defective connector.

STEP 6. M.U.T.-III data list
Item 9: Ignition switch power supply (Refer to data list reference table ).

Q. Is the check result normal?
Intermittent malfunction (Refer to GROUP 00 - How to Cope with Intermittent Malfunction ).
Replace the transfer-ECU.
