FUNCTION AND CONTROL BY ETACS-ECU


Following functions are controlled by ETACS-ECU.
Function
 Function description
Lamp reminder buzzer function
Door-ajar warning buzzer function
Turn-signal lamp buzzer function
Multi centre display buzzer function
Seat belt reminder function
Ignition key reminder buzzer function <Vehicles for GCC>
Speed warning buzzer function <Vehicles for GCC>
Sunroof buzzer function <Vehicles for South Africa>

CONTROL

LAMP REMINDER BUZZER FUNCTION

When the ignition key is removed and then the driver’s door is opened with turning ON the tail lamp or the headlamp, the buzzer sounds continuously to alert the driver that the lamp is still ON.
However, if the tail lamp or the headlamp is turned off by the headlamp automatic-shutdown off function, the buzzer does not sound.

DOOR-AJAR WARNING BUZZER FUNCTION (THE INITIAL CONDITION: WITH FUNCTION)

If the vehicle speed exceeds 8 km/h with the door-ajar, the buzzer sounds (beeps eight times) to alert the driver that the door is not properly closed.
note Availability of the door-ajar alarm function can be selected by the customise function (Refer to ).

TURN-SIGNAL LAMP BUZZER FUNCTION (THE INITIAL CONDITION: WITHOUT FUNCTION)

ETACS-ECU buzzer (sound like relay sound) sounds simultaneously with the blinks of the turn-signal lamp and the hazard lamp.
note Availability of the turn-signal lamp operation tone function can be selected by the customise function (Refer to ).

MULTI CENTRE DISPLAY BUZZER FUNCTION
When ETACS-ECU receives the buzzer sound requiring signal from the multi centre display, the buzzer sounds.

SEAT BELT REMINDER FUNCTION





  • When the ignition switch is turned ON with the driver’s seat belt unbuckled, the ETACS-ECU triggers a warning buzzer (repetition of a short beep sound) for 6 seconds to notify the driver that the seat belt is not fastened. The seat belt warning lamp illuminates while the buzzer is sounding.
  • When the ETACS-ECU detects that the vehicle speed signal sent from the combination meter through CAN communication exceeds 8 km/h and the driver’s seat belt buckle is not engaged, it will trigger a warning buzzer (repetition of 5-seconds ON and 2.8-seconds OFF for 12 cycles). The seat belt warning lamp flashes in synchronization with the buzzer’s sounding.
  • The ETACS-ECU stops the buzzer when the driver’s seat belt buckle is engaged while the warning cycle is ON.
  • The ETACS-ECU stops the buzzer when the ignition switch turned OFF while the warning cycle is ON.

IGNITION KEY REMINDER BUZZER FUNCTION <VEHICLES FOR GCC>

When the driver’s door is opened with the ignition key inserted in the ignition key cylinder (ignition switch is in the OFF position,) the buzzer sounds intermittently to indicate that the ignition key has not been removed.

SPEED WARNING BUZZER FUNCTION <VEHICLES FOR GCC>
If the vehicle speed exceeds 120 km/h, the buzzer sounds intermittently to warn the driver that the speed is too high.

SUNROOF BUZZER FUNCTION <VEHICLES FOR SOUTH AFRICA>
When the power window lock switch of a power window main switch operates a sunroof switch in the state of ON, ETACS-ECU sounds a buzzer.
