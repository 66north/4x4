Inspection Procedure 1: The security alarm is not armed (The security indicator does not illuminate).


caution Whenever the ECU is replaced, ensure that the input and output signal circuits are normal.










COMMENTS ON TROUBLE SYMPTOM
If this function does not work normally, the input signal circuit(s) below, the security indicator, or the ETACS-ECU may be defective. Also, the security alarm function may have been disabled with a customisation function.

  • Keyless entry transmitter
  • Key reminder switch
  • Ignition switch (ACC)
  • Hood switch

PROBABLE CAUSES

  • The security indicator is defective.
  • Malfunction of the ETACS-ECU
  • Damaged harness wires and connectors

DIAGNOSIS PROCEDURE

STEP 1. Check the operation of the keyless entry system.
Check that the keyless entry system works normally.

Q. Is the check result normal?
Go to Step 2.
Refer to GROUP 42 - keyless Entry System, Troubleshooting .

STEP 2. Pulse check
Check the input signals below, which are related to the ignition key reminder function.
System switch
 Check condition
Ignition switch (ACC)
 When turned from the LOCK (OFF) position to the ACC position.
Key reminder switch
 When the inserted ignition key is pulled out
Hood switch
 A hood opened
OK: The M.U.T.-III sounds or the voltmeter needle fluctuates.

Q. Are the check result normal?
Go to Step 3.
Refer to Input Signal Chart .

STEP 3. Connector check: D-40 security indicator connector

Q. Is the check result normal?
Go to Step 4.
Repair the defective connector.

STEP 4. Checking the security indicator
Refer to .

Q. Is the check result normal?
Go to Step 5.
Replace the security indicator.

STEP 5. Voltage measurement at the D-40 security indicator connector.
(1)Disconnect the connector, and measure at the wiring harness side.
(2)Voltage between D-40 security indicator connector terminal No.1 and body earth
OK: System voltage

Q. Is the check result normal?
Go to Step 7.
Go to Step 6.

STEP 6. Check the wiring harness between D-40 security indicator connector (terminal No.1) to fusible link (1).
note Prior to the wiring harness inspection, check intermediate connectors D-39 and D-241 and, and repair if necessary.

  • Check the power supply line for open circuit and short circuit.

Q. Is the check result normal?
The trouble can be an intermittent malfunction (Refer to GROUP 00 - How to Cope with Intermittent Malfunction ).
Repair the wiring harness.

STEP 7. Connector check: D-421 ETACS-ECU connector

Q. Is the check result normal?
Go to Step 8.
Repair the defective connector.

STEP 8. Check the wiring harness from D-421 ETACS-ECU connector terminal No.26 to D-40 security indicator connector terminal No.2.
note Prior to the wiring harness inspection, check intermediate connector D-39, and repair if necessary.

  • Check the input and output lines for open and short circuit.

Q. Is the check result normal?
Go to Step 9.
Repair the wiring harness.

STEP 9. Customise function by using the SWS monitor.
Use the SWS monitor customisation function to confirm that "Security alarm function" is set to "Horn/Hazard".
caution The SWS monitor must be used in order to confirm and change that setting. Use the SWS monitor to confirm and/or change the customised function.

Q. Is the check result normal?
Go to Step 10.
Use the SWS monitor customisation function to set "Security alarm function" is set to "Horn/Hazard" (Refer to ).

STEP 10. Retest the system.
Check that the security alarm works normally.

Q. Is the check result normal?
The trouble can be an intermittent malfunction (Refer to GROUP 00 - How to Cope with Intermittent Malfunction ).
Replace the ETACS-ECU.
