GENERAL INFORMATION



 Mitsubishi multi-communication system (multivision display) with built-in hard disk drive (40 gigabyte) has been installed. Also, a 7-inch liquid crystal display of wide 2 DIN size has been adopted.

  • The NAVTEQ map data (large volume map data for Europe) has been stored. <Except vehicle RUSSIA>
  • The Mitsubishi multi-communication system with NAVTEQ map data (large volume map data for RUSSIA) and the Mitsubishi multi-communication system without NAVTEQ map data are available. <Vehicle RUSSIA>
  • The 7-inch wide liquid crystal monitor has been adopted. The operation switch for operability has been equipped.
  • Most sophisticated LSI* has been used as one for car navigation system.
    note *:LSI stands for Large Scale Integration, and is a large-scale integrated circuit (IC) containing between 1,000 and 100,000 circuit elements.
  • It is compatible with nine languages (English, Germany, French, Spanish, Italian, Portuguese, Dutch, Swedish and Danish). <Except vehicle RUSSIA>
  • The multivision display for Russia corresponds to Russian only.<Vehicle RUSSIA>
  • Compatible with RDS function.
  • An exclusive tuner for TMC is incorporated.
  • Adopting the ROCKFORD™ ACOUSTIC DESIGN PREMIUM SOUND SYSTEM enables a great variety of sound and field settings through communication with the audio amplifier.
  • Control by steering wheel remote control switch
Screen (function)
 Contents
Navigation screen
Displays navigation functions such as road map, search, guidance and information retrieval.
Screen of current vehicle location
 Information on vehicle’s current position (such as latitude, longitude, altitude and GPS reception status)
CD/DVD
 Plays the CD or DVD inserted to the drive (Designed for MP3/WMA file)
Music server screen
 Plays music recorded in the hard disk drive and records a music CD. (The title whose information is acquired from CDDB can be displayed.)
Rear display screen
 Displays image via a DVD drive-incorporated rear seat 9-inch wide LCD display.
Sound control
 Adjusts the high/middle/low tone and balance. Function to set Equalizer type, sound field, and others when adopting the ROCKFORD™ ACOUSTIC DESIGN PREMIUM SOUND SYSTEM
Radio screen
 Displays available broadcast stations and channels. FM(RDS)/MW/LW
Rear view camera screen
 Displays rear view camera image.
