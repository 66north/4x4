Code No. P0638: Throttle Valve Control Servo Circuit Range/Performance Problem










OPERATION



  • Refer to P0657 - Throttle valve control servo relay circuit malfunction .
  • Refer to P2101 - Throttle valve control servo magneto malfunction .

FUNCTION



  • Engine-A/T-ECU checks the electronic controlled throttle system for abnormal conditions.

TROUBLE JUDGMENT


Check Conditions
  • Battery voltage is more than 8.3 V.
  • Throttle position sensor (main) output voltage is 0.35 - 4.8 V.
  • Drop of throttle position sensor (main) output voltage per 100 milliseconds is more than 0.04 V.
Judgement Criterion
  • Throttle position sensor (main) output voltage has continued to be above 0.5 V higher than the target throttle position sensor (main) voltage.
Check Conditions
  • Battery voltage is more than 8.3 V.
  • Throttle position sensor (main) output voltage is 0.35 - 4.8 V.
  • Ignition switch is in "ON" position.
  • Throttle position sensor is normal.
Judgement Criterion
  • Difference is more than 1 V between actual output voltage and target voltage of throttle position sensor (main) for 4 seconds.

PROBABLE CAUSES



  • Failed throttle valve return spring.
  • Failed throttle valve operation.
  • Failed throttle valve control servo.
  • Open/short circuit in throttle valve control servo connector contact.
  • Failed engine-A/T-ECU.

DIAGNOSIS
note
    *1: L.H. drive vehicles
    *2: R.H. drive vehicles
