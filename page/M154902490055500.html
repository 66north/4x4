Code No.05 Communication Error with Power Window Main Switch






















caution If diagnosis code No.05 is set in the ETACS-ECU, diagnose the CAN bus lines.
note When the diagnosis codes relating to the CAN communication error are set, make sure of the vehicle equipment. When the vehicle is not equipped with the system the SWS communicates to, the diagnosis code is always set. This is not abnormal.

TROUBLE JUDGMENT
The ETACS-ECU communicates with the power window main switch through the SWS communication lines. If there is any trouble in that communication, diagnosis code No.05 will be set.

COMMENTS ON TROUBLE SYMPTOM
If the power window main switch, the ETACS-ECU or the CAN bus line fails, this diagnosis code will be set.

PROBABLE CAUSES

  • Malfunction of the power window main switch
  • Malfunction of the ETACS-ECU switch
  • Malfunction of the CAN bus

DIAGNOSIS PROCEDURE

STEP 1. M.U.T.-III CAN bus diagnostics
Use the M.U.T.-III to diagnose the CAN bus lines.

Q. Is the check result normal?
Go to Step 2.
Repair the CAN bus line (Refer to GROUP 54D - Troubleshooting ).

STEP 2. Check whether the diagnosis code is reset.
Check again if the diagnosis code is set.
(1)Erase the diagnosis code.
(2)Ignition switch: LOCK (OFF) position to ON
(3)On completion, check that the diagnosis code is not reset.

Q. Is the diagnosis code set?
Go to Step 3.
Intermittent malfunction (Refer to GROUP 00 - How to use troubleshooting/Inspection Service Points ).

STEP 3. Pulse check
Check the input signal from the ignition switch.
System switch
 Check condition
Ignition switch (IG1)
 When turned from ACC to ON
OK: The M.U.T.-III sounds or the voltmeter needle fluctuates.

Q. Is the check result normal?
Go to Step 4.
Diagnose the ETACS (Refer to GROUP 54A - Input signal chart ).

STEP 4. Connector check: F-02 power window main switch connector

Q. Is the check result normal?
Go to Step 5.
Repair the defective connector.

STEP 5. Voltage measurement at the F-02 power window main switch connector.
(1)Disconnect the connector, and measure at the wiring harness side.
(2)Ignition switch: ON
(3)Voltage between F-02 power window main switch connector terminal No.6 and body earth
OK: System voltage

Q. Is the check result normal?
Go to Step 14.
Go to Step 6.

STEP 6. Connector check: D-408 power window relay connector

Q. Is the check result normal?
Go to Step 7.
Repair the defective connector.

STEP 7. Check the power window relay.
Refer to GROUP 42 - On-vehicle Service .

Q. Is the check result normal?
Go to Step 8.
Replace the power window relay.

STEP 8. Voltage measurement at D-408 power window relay connector.
(1)Remove the power window relay, and measure at the junction block side.
(2)Voltage between D-408 power window relay connector terminal No.5 and body earth
OK: System voltage

Q. Is the check result normal?
Go to Step 10.
Go to Step 9.

STEP 9. Check the wiring harness between D-408 power window relay connector terminal No.5 and fusible link (4).
note Prior to the wiring harness inspection, check intermediate connector D-238 and junction block connector D-403, and repair if necessary.

  • Check the power supply line for open circuit.

Q. Is the check result normal?
The trouble can be an intermittent malfunction (Refer to GROUP 00 - How to use troubleshooting/Inspection Service Points ).
Repair the wiring harness.

STEP 10. Resistance measurement at D-408 power window relay connector.
(1)Remove the power window relay, and measure at the junction block side.
(2)Resistance between D-408 power window relay connector terminal No.3 and body earth
OK: Continuity exists (2 Ω or less)

Q. Is the check result normal?
Go to Step 12.
Go to Step 11.

STEP 11. Check the wiring harness between D-408 power window relay connector terminal No.3 and body earth.
note Prior to the wiring harness inspection, check junction block connector D-405, and repair if necessary.

  • Check the earth wires for open circuit.

Q. Is the check result normal?
The trouble can be an intermittent malfunction (Refer to GROUP 00 - How to use troubleshooting/Inspection Service Points ).
Repair the wiring harness.

STEP 12. Connector check: D-419 ETACS-ECU connector

Q. Is the check result normal?
Go to Step 13.
Repair the defective connector.

STEP 13. Check the wiring harness between D-408 power window relay connector terminal No.4 and F-02 power window main switch connector terminal No.6.
note Prior to the wiring harness inspection, check intermediate connectors D-133 <LHD> or D-114 <RHD> and junction block connector D-403, and repair if necessary.

  • Check the power supply line for open circuit.

Q. Is the check result normal?
The trouble can be an intermittent malfunction (Refer to GROUP 00 - How to use troubleshooting/Inspection Service Points ).
Repair the wiring harness.

STEP 14. Resistance measurement at the F-02 power window main switch connector.
(1)Disconnect the connector, and measure at the wiring harness side.
(2)Resistance between F-02 power window main switch connector terminal No.2 and body earth
OK: Continuity exists (2 Ω or less)

Q. Is the check result normal?
Go to Step 16.
Go to Step 15.

STEP 15. Check the wiring harness between F-02 power window main switch connector terminal No.2 and body earth.
note Prior to the wiring harness inspection, check intermediate connectors D-133 <LHD> or D-114 <RHD>, and repair if necessary.

  • Check the earth wires for open circuit.

Q. Is the check result normal?
The trouble can be an intermittent malfunction (Refer to GROUP 00 - How to use troubleshooting/Inspection Service Points ).
Repair the wiring harness.

STEP 16. Connector check: D-420 ETACS-ECU connector

Q. Is the check result normal?
Go to Step 17.
Repair the defective connector.

STEP 17. Check the wiring harness between D-420 ETACS-ECU connector terminal No.59 and F-02 power window main switch connector terminal No.4.
note Prior to the wiring harness inspection, check intermediate connectors D-133 <LHD> or D-114 <RHD>, and repair if necessary.

  • Check the communication lines for open and short circuit.

Q. Is the check result normal?
Go to Step 18.
Repair the wiring harness.

STEP 18. Check whether the diagnosis code is reset.
Replace the power window main switch, and then check that the diagnosis code is not reset to the ETACS-ECU.
(1)Replace the power window main switch.
(2)Erase the diagnosis code.
(3)Ignition switch: LOCK (OFF) position to ON
(4)On completion, check that diagnosis code No.05 is not reset.

Q. Is diagnosis code No.05 set?
Replace the ETACS-ECU.
The procedure is complete.
