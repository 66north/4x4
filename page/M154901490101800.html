ECU CHECK


Use the M.U.T.-III and the SWS monitor to check ECUs.
1.The following ECUs can be checked by using the M.U.T.-III and the SWS monitor.
note If a malfunction is found by the "ECU check", proceed "Symptom Procedure" (Refer to ).
