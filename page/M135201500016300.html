Code No.51: Valve Relay ON Defective










OPERATION



  • The solenoid valve is always energised through the ABS-ECU internal valve relay.
  • The ABS-ECU activates the solenoid valve by turning ON its valve relay.

DIAGNOSIS CODE SET CONDITION


The solenoid valve remains to be energised when the ABS-ECU turned the valve relay to OFF. (Short in the solenoid valve circuit)

PROBABLE CAUSES


The most likely causes for these diagnosis codes to set are:

  • Damaged wiring harness and connector
  • Malfunction of the valve relay
  • Malfunction of ABS-ECU

DIAGNOSIS

STEP 1. Check the valve relay.
Refer to .

Q. Is the check result normal?
Go to Step 2.
Replace the valve relay. Then go to Step 7.

STEP 2. Voltage measurement at ABS-ECU connector D-250.
note Prior to the voltage inspection, check the ABS-ECU connector D-250 and the valve relay connector B-17X. Repair if necessary.
(1)Disconnect the connector D-250.
(2)Turn the ignition switch to the "ON" position.
(3)Measure the voltage between terminal No.37 and earth.
OK: System voltage

Q. Is the check result normal?
Go to Step 3.
Repair the wiring harness between the valve relay connector B-17X terminal No. 1 and the ABS-ECU connector D-250 terminal No. 37.

STEP 3. Inspect the wiring harness between the ABS-ECU connector D-250 and the valve relay connector B-17X.
caution When using the circuit tester for inspection, be sure to apply the voltage in a correct direction to the inspection terminal because the diode is mounted between the terminals.
note Prior to the voltage inspection, check the ABS-ECU connector D-250 and the valve relay connector B-17X, and repair if necessary.
ABS-ECU connector D-250 terminal No.50 (circuit tester: +) - valve relay connector B-17X terminal No. 2 (circuit tester: -)
OK: Continuity exists (2 Ω or less)

Q. Is the check result normal?
Go to Step 4.
Repair or replace it.

STEP 4. Check the harness wires between valve relay connector B-17X and earth.
note Prior to the voltage inspection, check the valve relay connector B-17X, and repair if necessary.
Valve relay connector B-17X terminal No.4 - earth
OK: Continuity exists (2 Ω or less)

Q. Is the check result normal?
Go to Step 5.
Repair or replace it.

STEP 5. Voltage measurement at valve relay connector B-17X.
note Prior to the voltage inspection, check the valve relay connector B-17X, and repair if necessary.
(1)Disconnect the connector B-15X.
(2)Turn the ignition switch to the "ON" position.
(3)Measure the voltage between terminal No.5 and earth.
OK: System voltage

Q. Is the check result normal?
Go to Step 6.
Repair or replace it.

STEP 6. Check whether the diagnosis code is reset.
Check again if the diagnosis code is set.
(1)Turn the ignition switch to the "ON" position.
(2)Erase the diagnosis code.
(3)Turn the ignition switch to the "LOCK" (OFF) position.
(4)Turn the ignition switch to the "ON" position.
(5)Check if the diagnosis code is reset.
(6)Turn the ignition switch to the "LOCK" (OFF) position.

Q. Is code No.51 set?
Replace the ABS-ECU. Then go to Step 7.
It can be assumed that this malfunction is intermittent. Refer to GROUP 00, How to Use Troubleshooting/Inspection Service Points - How to Cope with Intermittent Malfunction .

STEP 7. Check whether the diagnosis code is reset.
Check again if the diagnosis code is set.
(1)Turn the ignition switch to the "ON" position.
(2)Erase the diagnosis code.
(3)Turn the ignition switch to the "LOCK" (OFF) position.
(4)Turn the ignition switch to the "ON" position.
(5)Check if the diagnosis code is set.
(6)Turn the ignition switch to the "LOCK" (OFF) position.

Q. Is code No.51 set?
Repeat the troubleshooting from Step 1.
The procedure is complete.
