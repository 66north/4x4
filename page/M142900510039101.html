INSPECTION PROCEDURE B-5: The Power Window Timer Function does not work Normally. <Using SWS Monitor>


caution Whenever the ECU is replaced, ensure that the input and output signal circuits are normal.

OPERATION


The ETACS-ECU operates this function in accordance with the input signals below.

  • Ignition switch (IG1)
  • Driver’s door switch

COMMENTS ON TROUBLE SYMPTOM


If the power window timer function does not work normally, these input signal circuit(s) or the ETACS-ECU may be defective.

POSSIBLE CAUSES



  • Malfunction of the driver’s door switch
  • Malfunction of the ETACS-ECU
  • Damaged harness wires and connectors

DIAGNOSTIC PROCEDURE


STEP 1. ECU check by using the SWS monitor


Check that the power supply and earth lines to the ETACS-ECU and the power window main switch (power window module) and the SWS communication lines are normal.

  • Turn the ignition switch from the ON to the OFF position.
ECUS TO BE CHECKED
  • ETACS ECU
  • P/W MODULE
OK: "OK" are displayed for all the items
note "OK" is displayed for the power window switch (power window module) for approximately one minute, and then changed to "NG". However, only "OK" is displayed for the ETACS-ECU.

Q. Are the check result normal?

YES ("OK" are displayed for all the items) : Go to Step 2.

NO ("NG" is displayed on the "ETACS ECU" menu.) : Refer to GROUP 54C, Inspection Procedure 2 "Communication with the ETACS-ECU is not possible ."

NO ("NG" is displayed on the "P/W MODULE" menu.) : Refer to GROUP54C, Inspection Procedure 5 "Communication with the power window main switch (power window module) is not possible ."

STEP 2. Function diagnosis by using the SWS monitor


Check the SWS communication signal, which are related to the power window timer function.

Q. Are the check result normal?

YES <Normal conditions are displayed for all the items.> : Go to Step 3.

NO <Normal condition is not displayed for item No.30> : Refer to GROUP 54A, inspection procedure C-2 "The ignition switch (IG1) signal is not received ."

NO <Normal condition is not displayed for item No.33> : Replace the ETACS-ECU.

NO <Normal condition is not displayed for item No.71> : Replace the power window main switch.

Q. Is the check result normal?

Go to Step 4.

Refer to GROUP 54A, Inspection Procedure C-5 "The driver’s door switch signal is not received <LH drive vehicles> or <RH drive vehicles> ."

STEP 4. Retest the system.


Check that the power window timer function works normally.

Q. Is the check result normal?

The trouble can be an intermittent malfunction (Refer to GROUP 00, How to Use Troubleshooting/Inspection Service Points - How to Cope with Intermittent Malfunction ).

Replace the power window main switch.
