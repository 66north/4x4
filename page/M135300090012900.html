DIAGNOSTIC FUNCTION


ASTC-ECU has the following functions for easier system checks. The following items can be diagnosed using M.U.T-III.

  • Diagnosis code set (Refer to ).
  • Service data output (Refer to ).
  • Actuator test (Refer to ).

FAIL-SAFE DIAGNOSIS FUNCTION


If any malfunction is detected by the self-diagnostic function, the ASTC-ECU illuminates the ABS warning lamp, brake warning lamp*, active stability control system ON indicator, active traction control system ON indicator and active stability control OFF indicator and it disables ABS, EBD, ASTC control.
note *: The brake warning lamp is used as EBD control warning lamp.
