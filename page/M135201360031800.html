Code No.21: Wheel Speed Sensor (FR) Control Phase Time Exceeded<br />Code No.22: Wheel Speed Sensor (FL) Control Phase Time Exceeded<br />Code No.23: Wheel Speed Sensor (RR) Control Phase Time Exceeded<br />Code No.24: Wheel Speed Sensor (RL) Control Phase Time Exceeded














OPERATION

  • A toothed rotor generates a voltage signal as it moves across the pickup field of each wheel speed sensor.
  • The amount of voltage generated at each wheel is determined by the clearance between the rotor teeth and the wheel speed sensor, and by the speed of rotation.
  • The wheel speed sensors transmit the frequency of the voltage signal to the ABS-ECU.

DIAGNOSIS CODE SET CONDITIONS
Diagnosis codes No. 21, 22, 23, and 24 are set in the following cases.

  • Although no open circuit is found, no input signal from the wheel speed sensors of one or more wheels is received.
  • Any abnormality is found with the input signals from the wheel speed sensors.

PROBABLE CAUSES

  • Malfunction of the wheel speed sensor
  • Damaged wiring harness or connector
  • Malfunction of the rotor
  • The gap between the wheel speed sensor and the rotor is excessive.
  • Malfunction of the wheel bearing
  • Malfunction of the ABS-ECU

DIAGNOSIS

STEP 1. Check for wheel speed sensor installation
Check how the wheel speed sensor is installed (Disconnection of wheel speed sensor, loose mounting bolt, etc)..

Q. Is the check result normal?
Go to Step 2.
Reinstall the wheel speed sensor correctly. Then go to Step 14.

STEP 2. Inspect the wheel speed sensor.
Check the wheel speed sensor relevant to the diagnosis code. For the applicable inspection procedure. (Refer to .)

Q. Is the check result normal?
YES <wheel speed sensor (FR)> : Go to Step 5.
YES <wheel speed sensor (FL)> : Go to Step 6.
YES <wheel speed sensor (RR)> : Go to Step 7.
YES <wheel speed sensor (RL)> : Go to Step 8.
Go to Step 3.

STEP 3. Check the wheel bearing for looseness.
note If the wheel bearing is loose, the gap between the wheel speed sensor and rotor may become excessive. Check the hub assembly, for looseness.
When the code No.21 is set
  • Check the front hub assembly <Refer to GROUP 26, Hub and Knuckle Assembly - Inspection (RH) >.
When the code No.22 is set
  • Check the front hub assembly <Refer to GROUP 26, Hub and Knuckle Assembly - Inspection (LH) >.
When the code No.23 is set
  • Check the rear hub assembly <Refer to GROUP 27, Rear Axle Hub Assembly - Inspection (RH) >.
When the code No.24 is set
  • Check the rear hub assembly <Refer to GROUP 27, Rear Axle Hub Assembly - Inspection (LH) >.

Q. Is the check result normal?
Go to Step 4.
NO (front) : Replace the front hub assembly (Refer to GROUP 26, Hub and knuckle assembly disassembly and reassembly ). Then go to Step 14.
NO (rear) : Replace the rear hub assembly (Refer to GROUP 27, knuckle removal and installation ). Then go to Step 14.

STEP 4 Check the rotor.
Check the rotor, for foreign material or deformation.
When the code No.21 is set
  • Check the rotor <FR> (Refer to ).
When the code No.22 is set
  • Check the rotor <FL> (Refer to ).
When the code No.23 is set
  • Check the rotor <RR> (Refer to ).
When the code No.24 is set
  • Check the rotor <RL> (Refer to ).

Q. Is the check result normal?
YES <wheel speed sensor (FR)> : Go to Step 5.
YES <wheel speed sensor (FL)> : Go to Step 6.
YES <wheel speed sensor (RR)> : Go to Step 7.
YES <wheel speed sensor (RL)> : Go to Step 8.
NO (front) : If the rotor is contaminated with foreign material, clean it. If the front hub assembly is deformed, replace it (Refer to GROUP 26, Driveshaft Assembly ). Then go to Step 14.
NO (rear) : If the rotor is contaminated with foreign material, clean it. If the rear hub assembly is deformed, replace it (Refer to GROUP 27, Driveshaft Assembly ). Then go to Step 14.

STEP 5. Connector check: D-256 ABS-ECU connector, A-03 intermediate connector and A-36 wheel speed sensor (FR) connector.

Q. Is the check result normal?

Repair or replace the damaged component(s). Then go to Step 14.

STEP 9. Check the continuity between D-256 ABS-ECU connector and A-36 wheel speed sensor (FR) connector.
(1)Disconnect the D-256 ABS-ECU connector and A-36 wheel speed sensor (FR) connector, and then measure the resistance at the D-256 ABS-ECU connector and A-36 wheel speed sensor (FR) connector side.
(2)Check the continuity between the following terminals.

  • D-256 ABS-ECU connector terminal No.10 - A-36 wheel speed sensor (FR) terminal No.1
  • D-256 ABS-ECU connector terminal No.23 - A-36 wheel speed sensor (FR) terminal No.2
OK: Continuity exists (2 Ω or less)

Q. Is the check result normal?
Go to Step 13.
Repair the wiring harness. Then go to Step 14.

STEP 13. Check whether the diagnosis code is reset.
Check again if the diagnosis code is set.
(1)Drive the vehicle with speed exceeds 8 km/h to recognize wheel signals to the ABS-ECU.
(2)Erase the diagnosis code.
(3)Turn the ignition switch to the "LOCK" (OFF) position.
(4)Turn the ignition switch to the "ON" position.
(5)Disconnect the M.U.T.-III.
(6)Drive the vehicle with speed exceeds 40 km/h to recognize wheel signals to the ABS-ECU.
(7)Check if the diagnosis code is reset.
(8)Turn the ignition switch to the "LOCK" (OFF) position.

Q. Is code No.21, 22, 23 or 24 set?
Replace the ABS-ECU. Then go to Step 14.
Intermittent malfunction (Refer to GROUP 00 - How to Cope with Intermittent Malfunction ).

STEP 14. Check whether the diagnosis code is reset.
Check again if the diagnosis code is set.
(1)Drive the vehicle with speed exceeds 8 km/h to recognize wheel signals to the ABS-ECU.
(2)Erase the diagnosis code.
(3)Turn the ignition switch to the "LOCK" (OFF) position.
(4)Turn the ignition switch to the "ON" position.
(5)Disconnect the M.U.T.-III.
(6)Drive the vehicle with speed exceeds 40 km/h to recognize wheel signals to the ABS-ECU.
(7)Check if the diagnosis code is reset.
(8)Turn the ignition switch to the "LOCK" (OFF) position.

Q. Is code No.21, 22, 23 or 24 set?
Repeat the troubleshooting from Step 1.
The procedure is complete.
