FAIL-SAFE/FAIL-PROOF FUNCTION AND DIAGNOSIS



  • In the event of a fault in the ECU or any of the sensors and actuators, or in the event of seriously erroneous operation on the part of the driver, a fail-safe/fail-proof function controls the A/T as appropriate to protect the passengers and vehicle from danger.
  • The ECU constantly monitors inputs from sensors and outputs from actuators. In the event of an abnormality in these signals, it memorizes the symptoms and diagnoses the abnormality.

FAIL-SAFE/FAIL-PROOF FUNCTION



  • In the event of a fault in the electronic control system, the vehicle can still be operated. In the D range, the vehicle can be driven in 3rd gear. In the sport mode, it can be driven in 2nd gear (except if the diagnosis code is P1773 [P0753] - P1778 [P0743], P1779 [P0731] - P1784 [P0736], P1788 [P1751]).
  • During forward motion at speeds of 7 km/h and above, the reverse gear cannot be engaged even if the selector lever is moved to the "R" position.

DIAGNOSIS FUNCTION


(1) Diagnosis Codes
    i) Diagnosis Items and Fail-Safe Items Among the diagnosis codes, higher values represent fail-safe items.

    • Diagnosis Items: Code Nos.P1763 [P0713], P1764 [P0712], P1769, P1770 [P0705], P1771 [P0705], P1786 [P0741], P1787 [P0742]
    • Fail-safe items: Code Nos.P1766 [P0715], P1767 [P0720], P1773 [P0753] - P1778 [P0743], P1779 [P0731] - P1784 [P0736], P1788 [P1751]
    ii) Indication Method and Sequence

      a) During Normal Vehicle Operation
      If the fail-safe function operates during normal vehicle operation, the N range lamp flashes at a frequency of 1 Hz to inform the driver (This occurs only in the D range).
      b) Display Sequence
      Diagnosis codes are indicated in the order in which they were issued, with fail-safe items followed by diagnosis items. This sequence is then repeated.
    iii) Memorization

    • Up to 10 diagnosis items can be memorized.
    • If the memory capacity is exceeded, diagnosis and fail-safe items in the memory are overwritten, starting with the oldest.
    • No code can be memorized more than once.
    iv) Diagnosis Code Deletion

      a) Automatic Deletion All diagnosis codes are deleted from memory the 200th time the A/T fluid temperature reaches 50°C after memorization of the most recent diagnosis code.
      b) Forced Deletion Memorized diagnosis codes can be deleted using the M.U.T.-III provided the following conditions are satisfied:

      • The ignition switch is ON;
      • There is no detection pulse from the input shaft speed sensor;
      • There is no detection pulse from the output shaft speed sensor;
      • The inhibitor switch should be P position;
      • The fail-safe function is not operational.
