HOW TO PERFORM CHASSIS NUMBER (CHASSIS NO.) WRITING <4M41>


Write the Chassis Number (Chassis No.) of immobilizer system as following procedure.
The Chassis Number (Chassis No.) is stored in the engine-ECU or immobilizer-ECU by the vehicle manufacture. If the Chassis No. to be stored in the engine-ECU or immobilizer-ECU is eliminated fraudulently, the engine warning lamp illuminates and Diagnostic Trouble Code (diagnosis code) No.P0630 (Chassis No. malfunction) is shown. When the engine-ECU or immobilizer-ECU is replaced, and entry of the Chassis No. necessary due to diagnosis code No. P0630 (Chassis No. malfunction). Enter the Chassis No. in accordance with the procedure as follows:


CHASSIS NUMBER REGISTRATION PROCEDURE FOR ENGINE-ECU


caution
  • Check that diagnosis code No.P0603 (EEPROM malfunction) is not set. If diagnosis code No.P0603 (EEPROM malfunction) is set, entered Chassis No. cannot be stored. Therefore, carry out troubleshooting and repair the malfunction when this code is set.
  • To prevent damage to M.U.T.-III, always turn the ignition switch to the "LOCK" (OFF) position before connecting or disconnecting M.U.T.-III.
Connect the M.U.T.-III to the 16-pin diagnosis connector, and read the diagnosis code.
note For details on how to use the M.U.T.-III, refer to the "M.U.T.-III operation manual."
1.Ensure that the ignition switch is at the "LOCK" (OFF) position.
2.Start up the personal computer.
3.Connect special tool M.U.T.-III USB cable (MB991827) to special tool V.C.I. (MB991824) and the personal computer.
4.Connect M.U.T.-III main harness A (MB991910) to the V.C.I.
5.Connect the M.U.T.-III main harness A to the diagnosis connector of the vehicle.
6.Turn the V.C.I. power switch to the "ON" position.
note When the V.C.I. is energized, the V.C.I. indicator lamp will be illuminated in a green colour.
7.Start the M.U.T.-III system on the personal computer and turn the ignition switch to the "ON" position.

8.Select "IMMOBILIZER" in the "System Select" tag. Select the model year and push the OK button (Check each model if there is an option). Select "Special Function" on the next screen, and select "Engine Key Code Reg. & Chassis No. Writing" on the further screen.

9.Push the OK button after "Key Code Registration" is displayed.
10.Push the OK button after "Completed. Press the OK button and move to Chassis No. writing function." is displayed.

11.Enter the Chassis No. of registering vehicle and push the OK button.
12.Push the OK button after "Chassis No. Writing will start. Are you sure?" is displayed.
13.Return to the previous screen and "In Progress" is displayed at the lower-left corner on the screen.
14.Push the OK button after "Completed." is displayed.

15.Chassis No. writing result is displayed.
16.Complete the M.U.T.-III.
17.Disconnecting the M.U.T.-III is the reverse of the connecting sequence, making sure that the ignition switch is at the "LOCK" (OFF) position.

CHASSIS NUMBER REGISTRATION PROCEDURE FOR IMMOBILIZER-ECU.


Connect the M.U.T.-III to the 16-pin diagnosis connector, and read the diagnosis code.
note For details on how to use the M.U.T.-III, refer to the "M.U.T.-III operation manual."
1.Ensure that the ignition switch is at the "LOCK" (OFF) position.
2.Start up the personal computer.
3.Connect special tool M.U.T.-III USB cable (MB991827) to V.C.I. (MB991824) and the personal computer.
4.Connect M.U.T.-III main harness A (MB991910) to the V.C.I. (MB991824).
5.Connect the M.U.T.-III main harness A (MB991910) to the diagnosis connector.
6.Turn the V.C.I. (MB991824) power switch to the "ON" position.
note When the V.C.I. (MB991824) is energized, the V.C.I. (MB991824) indicator lamp will be illuminated in a green colour.
7.Start the M.U.T.-III system on the personal computer and turn the ignition switch to the "ON" position.

8.Select "IMMOBILIZER" in the "System Select" tag. Select the model year and push the OK button (Check each model if there is an option). Select "Coding" on the next screen, and select "Chassis No. Writing" on the further screen.

9.Enter the Chassis No. of registering vehicle and push the OK button.
10.Push the OK button after "Chassis No. Writing will start. Are you sure?" is displayed.
11.Return to the previous screen and "In Progress" is displayed at the lower-left corner on the screen.
12.Push the OK button after "Completed." is displayed.

13.Chassis No. writing result is displayed.
14.Select "Chassis No. Information" on the "Coding" screen after the Chassis No. writing is completed, and check that the Chassis No. is changed.
15.Resister the other ID code. (Refer to GROUP 54A, Immobilizer system - Key ID Registration Criteria Table <Diesel powered vehicle>.)
