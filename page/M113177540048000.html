Code No. P0134: Right Bank Oxygen Sensor (Front) Circuit No Activity Detected












OPERATION



  • Refer to Code No. P0130: Right bank oxygen sensor (front) circuit .
  • Refer to Code No. P0201: No. 1 Injector System .
  • Refer to Code No. P0203: No. 3 injector system .
  • Refer to Code No. P0205: No. 5 injector system .

FUNCTION



  • The engine-A/T-ECU effects air/fuel ratio feedback control in accordance with the signals from the right bank oxygen sensor (front).
  • If the right bank oxygen sensor (front) has deteriorated, corrections will be made by the right bank oxygen sensor (rear).
  • Code No. P0134 becomes stored in memory if a failure is detected in the above air/fuel ratio feedback control system.

TROUBLE JUDGMENT


Check Conditions
  • 20 seconds have passed since the engine starting sequence was completed.
  • Engine coolant temperature is more than approximately 76°C.
  • Engine speed is more than 1,200 r/min.
  • Volumetric efficiency is more than 30 %.
  • Throttle position sensor output voltage is more than 4 V.
  • Except while fuel is being shut off.
Judgment Criterion
  • Right bank oxygen sensor (front) output voltage does not get across approximately 0.5 V for 30 seconds.

PROBABLE CAUSES



  • Failed right bank oxygen sensor (front)
  • Exhaust leak
  • Incorrect fuel pressure
  • Failed engine-A/T-ECU
